## 셋업 방법

- 클론: `$ git clone https://github.com/soonitoon/coding-test`
- 레포지토리 이동: `$ cd ./coding-test`
- 배포(자동 커밋 & 푸쉬): `$ yarn run publish`
- 새 문서 템플릿 생성: `$ yarn run template`
